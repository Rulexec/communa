Code Hardcorius

Я давно хотел изложить свои знания относительно программирования, быть может они кому-нибудь могут помочь.

Так появился блог Code Hardcorius.

Лично для меня программирование это больше, чем продумывание архитектуры и написание кода. Для меня это стиль жизни, взгляд на мир, можно сказать, даже что-то около философии.

Мне понравилась эта цитата:

Математики думают, что Бог в уравнениях, нейрологи уверены, что Бог в мозге, а программисты уверены, что Бог — один из них. Говорят, Морган Фриман, но я не уверен. Очень не уверен.
